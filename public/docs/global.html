# Chip(props)

An interactive chip component with animation support. When selected, displays a gradient background with a smooth animation. Supports both light and dark themes. Animation Logic: 1. Each chip has a gradient background (motion.span) with the same layoutId="pill-tab" 2. When you click a chip, it becomes "selected" and the previous one becomes "unselected" 3. Because they share the same layoutId, Framer Motion treats them as the same element 4. This makes the gradient background smoothly animate from the old position to the new position 5. A spring transition is used to give it a natural, bouncy feel
Parameters:
Name Type Attributes Default Description
props Object The component props
text string The text to display inside the chip
selected boolean <optional>
 false Whether the chip is in a selected state
onClick function <optional>
 Optional click handler for the chip
